# 보고서{#reports}

사용 보고서는 평균 세션 길이, 일일 및 월별 참여 사용자 수, 업그레이드 및 기타 정보를 표시하여 앱 사용 방법과 빈도를 이해하는 데 도움을 줍니다. 유지 보고서는 초기 실행 후에 사용자 유지 수준 1일, 7일 및 28일을 빠르게 표시합니다.
